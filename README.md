## Installation

1. Download and install NodeJS on your machine: https://nodejs.org/en
2. Clone repository.

## Running in development.

Start server:

```bash
npm run dev
```

## Deploy

Build project:

```bash
npm run build
```

Preview

```bash
npm run preview
```
